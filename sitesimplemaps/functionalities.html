SimpleMaps Features & Functions

Based on the defined purpose, the functional requirements were identified, which represent the functionality needed for project implementation:

  1. - Search for an address on the map
  2. - Mark a point on the map
  3. - Mark an area on the map (circle, triangle and rectangle)
  4. - Insert a picture on the map (link and upload an image)
  5. - Insert an audio file on the map
  6. - Insert a video on the map
  7. - Edit / Delete an object
  8. - Create / Edit a map
  9. - Change a map

Features & Functions Implemented:

  1. - Search for an address on the map
  2. - Mark a point on the map
  3. - Mark an area on the map (triangle and rectangle)
  4. - Insert a picture on the map (link an image)
  5. - Insert a video on the map
  6. - Create a new map
  7. - Change a map
