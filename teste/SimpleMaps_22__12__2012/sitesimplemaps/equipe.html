Equipe SimpleMaps

Este projeto começou a ser desenvolvido pelos estudantes do 4° semestre do curso de Análise e Desenvolvimento de Sistemas (ADS) da Faculdade de Tecnologia Bandeirantes (BandTec) durante o primeiro semestre de 2012. Neste ponto os estudantes envolvidos no projeto eram: Danielle, Igor, Kelvin, Neudecir e Sidmar.

No segundo semestre de 2012 o projeto se tornou maior e os estudantes do 4° semestre do curso de Banco de Dados da BandTec: Andre, Bruno, Marcelo e Susana, foram envolvidos para apoiar os alunos do 5° semestre do curso de ADS, gerenciando o banco de dados que suporta a aplicação.
