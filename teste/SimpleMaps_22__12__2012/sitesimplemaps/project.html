The SimpleMaps Project

SimpleMaps is an academic project that was set up with the aim of creating a collaborative system accessed through the internet, in which users can search for geographic locations and interact with maps, inserting multimedia objects that represent their emotions and perceptions from places they usually go to. Such objects can be images, videos, audio files, text or even pictures on the map. By inserting these artifacts on the map, they should be geo-coded, which means, each object must have a latitude and longitude value associated. Link to SimpleMaps Project: http://projetos.bandtec.com.br/
