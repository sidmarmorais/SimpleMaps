SimpleMaps Features & Functions

Based on the defined purpose, the functional requirements were identified, which represent the functionality needed for the project implementation:

  1. - Search for an address on the map
  2. - Mark a point on the map
  3. - Mark an area on the map (circle, triangle and rectangle)
  4. - Insert a picture on the map (image’s URL and upload an image file)
  5. - Insert an audio file on the map (upload an audio file)
  6. - Insert a video on the map (video’s URL)
  7. - Edit / Delete an object
  8. - Create / Edit a map
  9. - Change a map

Features & Functions Implemented:

  1. - Search for an address on the map
  2. - Mark a point on the map
  3. - Mark an area on the map (triangle and rectangle)
  4. - Insert a picture on the map (image’s URL)
  5. - Insert a video on the map (video’s URL)
  6. - Create a new map
