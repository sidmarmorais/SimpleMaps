Funcionalidades SimpleMaps

Com base no propósito definido, os requisitos funcionais foram identificados e representam as funcionalidades necessárias para a implementação do projeto:

  1. - Buscar uma localidade no mapa
  2. - Marcar um ponto no mapa
  3. - Marcar uma área no mapa (círculo, triângulo e retângulo)
  4. - Inserir uma imagem no mapa (URL da imagem ou upload de arquivo de imagem)
  5. - Inserir um arquivo de áudio no mapa (upload de arquivo de áudio)
  6. - Inserir um vídeo no mapa (URL do vídeo)
  7. - Editar / Deletar um objeto
  8. - Criar / Editar um mapa
  9. - Trocar de mapa

Funcionalidades Implementadas:

  1. - Buscar uma localidade no mapa
  2. - Marcar um ponto no mapa
  3. - Marcar uma área no mapa (triângulo e retângulo)
  4. - Inserir uma imagem no mapa (URL da imagem)
  5. - Inserir um vídeo no mapa (URL do vídeo)
  6. - Criar um mapa
  7. - Trocar de mapa

Funcionalidades Pendentes (Roadmap) Requisitos Funcionais:
 
  1. - Marcar uma área no mapa (circulo)
  2. - Inserir uma imagem no mapa (upload de arquivo de imagem)
  3. - Inserir um arquivo de áudio no mapa (upload de arquivo de áudio)
  4. - Editar um mapa
  5. - Editar / Deletar um objeto
Requisitos Não-Funcionais:
 
  1. - Portar a aplicação para o sistema operacional Android
  2. - Disponibilizar gadgets para o Facebook
